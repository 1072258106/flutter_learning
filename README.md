# flutter_learning

一个用于学习flutter基础控件的demo

###包含以下内容
####按钮
RaisedButton、FlatButton、OutlineButton、DropdownButton、IconButton以及iOS风格的按钮
####布局
ListView布局、GridView布局
####输入框
TextField
####对话框
####图片资源
从本地获取图片、从网络获取图片、使用裁剪元素对图片进行裁剪
